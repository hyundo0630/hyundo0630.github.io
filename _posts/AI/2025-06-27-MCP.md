# Windows 환경 MCP 서버 구축하기.

## 1단계. Node.Js Install
### MCP 도구들이 대부분 Java Scripts 기반으로 만들어 졌기 때문에 Node.Js 를 설치해 줍니다.
- URL : https://nodejs.org/ko/download

설치가 완료 되셨다면, `node --version` 명령어를 통해 버전 정보가 출력 되는지 확인

```bash
$ node --version
```
